# FrameLab Front-end

Le Front-end de FrameLab.

## Infos pratiques

### Liste des liens

* `/` : Page d’accueil.
* `/login` : Page de connexion, et création de compte.
* `/register` : Page de création de compte.
* `/account` : Page de modification des informations du compte.
* `/admin` : Page de gestion et de création des challenges.

> **Note :** L'accès à la page `/admin` se fait uniquement via une saisie manuelle dans la barre d'adresse du navigateur. Les utilisateurs non authentifiés ou ne possédant pas les droits requis seront automatiquement redirigés ou verront leurs interactions ignorées.
